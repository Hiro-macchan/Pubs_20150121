  • \( Odds = \frac{a/a+b}{b/a+b} = \frac{a}{b} \)
  • \[ \hat{RD} = \frac{c}{c+d} \]
  • \[ \hat{RR} = \frac{a}{a+b} / \frac{c}{c+d} \]
  • \[ \hat{OR} = \frac{a}{b} / \frac{c}{d} \]

\( P_a = \frac{a}{a+b} \), \( P_c = \frac{c}{c+d} \)

  • (Rusk Difference) \[ RD = \hat{RD} \pm 1.96SE \] \[ SE = \sqrt{\frac{P_a(1-P_a)}{a+b}+\frac{P_c(1-P_c)}{c+d}} \]

  • (Risk Ratio) \[ log(RR) = log(\hat{RR}) \pm 1.96SE \] \[ SE=\sqrt{\frac{1}{a} - \frac{1}{a+b} + \frac{1}{c} - \frac{1}{c+d}} \]

  • (Odds Ratio) \[ log(OR) = log(\hat{OR}) \pm 1.96SE \]\[ SE=\sqrt{\frac{1}{a} + \frac{1}{b} + \frac{1}{c} + \frac{1}{d}} \]
